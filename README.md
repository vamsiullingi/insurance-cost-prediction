🏥 Insurance Premium Prediction & Risk AnalysisThis project aims to predict health insurance premiums using machine learning while providing actionable health insights through a Tableau BI dashboard.📝 Problem StatementInsurance providers often struggle to balance competitive pricing with risk management. This project uses historical data of 986 policyholders to determine how age, BMI, and chronic conditions influence premium costs, building a predictive engine to automate pricing for new customers.📊 Project WorkflowEDA & Visualization (Tableau): Analyzed key drivers of cost, identifying Age and Risk Score as primary influencers.Hypothesis Testing: Conducted T-tests and ANOVA to statistically prove that Chronic Diseases and Number of Surgeries significantly impact pricing ($p < 0.05$).Feature Engineering: Created BMI and RiskScore (a weighted sum of 6 health conditions) to enhance model accuracy.Machine Learning: Trained Linear Regression, Random Forest, and Gradient Boosting models.Deployment: Developed a Streamlit web application for real-time premium estimation.
